## Sample Shopify Integration

Integrate Nordbrains` Shopify app effortlessly.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Install via npm
  npm install nordbrains-shopify

  // Initialize client
  import { NordbrainsClient } from 'nordbrains-shopify';
  const client = new NordbrainsClient({ shop: 'your-shop.myshopify.com', apiKey: 'YOUR_API_KEY' });

  // Fetch products
  const products = await client.products.list();
  console.log(products);
  ```
  ```python
  # Install via pip
  pip install nordbrains-shopify

  # Initialize client
  from nordbrains_shopify import NordbrainsClient
  client = NordbrainsClient(shop='your-shop.myshopify.com', api_key='YOUR_API_KEY')

  # Fetch products
  products = client.products.list()
  print(products)
  ```
</CodeGroup>
