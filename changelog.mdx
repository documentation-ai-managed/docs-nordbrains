<Update label="2024-10-15" description="v2.0.0" tags={["feature", "improvement"]}>

## New Features

- Added support for Shopify Plus headless commerce setups, enabling fully decoupled storefronts with Hydrogen framework.
- Introduced mobile app prototyping with React Native for iOS and Android cross-platform development.

## Improvements

- Reduced custom web deployment time by `>50%` using optimized CI/CD pipelines.
- Enhanced e-commerce analytics integration with Google Analytics 4 and custom dashboards.

## Bug Fixes

- Fixed intermittent webhook failures in Shopify apps during high-traffic periods.

</Update>

<Update label="2024-09-20" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- New integration capabilities for ERP systems like SAP and custom APIs with `https://api.example.com/v1/integrations`.
- Client feedback implementation: Added multi-language support for web apps using Next.js i18n.

## Bug Fixes

- Resolved cart abandonment tracking issues in Shopify themes.
